<p><a name="beaten"></a></p>

<h4 id="beaten">Beaten</h4>

<ol>
<li>Persona 4 Golden (Vita)
<ul>
<li>Wow. Just… wow.</li>
<li>Made it to # 2 on my <a href="/favorite-games/#persona-4">Favorite Games of All Time list</a>.</li>
</ul>
</li>
<li>Steamworld Dig (PS4/Vita)
<ul>
<li>Fun game</li>
<li>Too short :(</li>
</ul>
</li>
<li>Risk of Rain (Vita)
<ul>
<li>Nice replayable portable title.</li>
</ul>
</li>
<li>Night in the Woods (PS4)
<ul>
<li>Easy contender for Indie Game of the Year.</li>
<li>If I hadn’t played Persona 4 Golden, this would be my
favorite game I’ve played in 2017.</li>
</ul>
</li>
<li>Nioh (PS4)
<ul>
<li>Was very good early on, but later became less interesting
as I found the available skills quite limited.</li>
<li>Story was… meh</li>
<li>Feels like a solid C+ / B-</li>
</ul>
</li>
<li>Horizon: Zero Dawn (PS4)
<ul>
<li>70 hours and loved every minute of it.</li>
<li>Solid A / A+</li>
</ul>
</li>
<li>Persona 5 (PS4).
<ul>
<li>97hrs 40min and it was quite good.</li>
<li>Makoto is bae <em>#waifu</em>.</li>
<li>Great game with excellent characters.</li>
<li>Solid A / A+.</li>
</ul>
</li>
<li>Deus Ex: Mankind Divided (PS4)
<ul>
<li>Story was alright. I liked that it connects to Deus Ex (2000).</li>
<li>Gameplay was decent.</li>
<li>Framerate on the Pro was rough.</li>
<li>Feels like a C.</li>
</ul>
</li>
</ol>
